API Documentaion

Resources

Users

  • _id: (by defult)
  • name: string(),min(2),max(99) ,required()
  • pass: string(),min(2),max(99),required()
  • email: string(),min(1),max(999) ,required(),email()
  • date_created: (by defult)
  • role: (by defult)

Vehicle

  • _id: (by defult)
  • name: string(),min(2),max(99) ,required()
  • info: string(),min(2),max(9999) ,required()
  • category: string(),min(1),max(99) ,required()
  • img_url: string(),min(1),max(9999) ,required()
  • price : number(),min(0),max(9999999) ,required()
  • date_created: (by defult)
  • user_id: (from the user token)

API Guide

STEP 1

For using my web-server you must to register first

Go register in routes post

STEP 2

After sign up do login

Go login in routes post

STEP 3

After the login you Should get a token
with this token you can see all of your vehicles's user's details ,add vehicles to the DB ,delete vehicles

Start work with my web-server

Routes

  • https://project-benor-2021.herokuapp.com/users/myinfo
    To get your info after login with token
  • https://project-benor-2021.herokuapp.com/vehicles/searchMyInfo/?s
    To search in my vehicles by name or info
  • https://project-benor-2021.herokuapp.com/vehicles/search/?s=honda
    To search in DB vehicles by name or info
  • https://project-benor-2021.herokuapp.com/vehicles/cat/car
    search by category name
  • https://project-benor-2021.herokuapp.com/vehicles/search/?s=
    search by name of vehicles or info
  • https://project-benor-2021.herokuapp.com//vehicles/cat/car/?page=1=
    show all vehicles from DB in page 1
  • https://project-benor-2021.herokuapp.com//vehicles/cat/car/?page=1&perPage=5
    show all vehicles from DB in page 1, 5 objects in page
  • https://project-benor-2021.herokuapp.com//vehicles/cat/car/?page=1&perPage=5&sort=price
    show all car's user in page 1, 5 objects in page sorting by price desc (From the big to the small)

  • https://project-benor-2021.herokuapp.com/users/register
    to sign up with { "name": "name", "pass": "pass", "email": "email@walla.com" } in the body of the POST req
  • https://project-benor-2021.herokuapp.com/users/login
    to login and get token { "pass": "pass", "email": "email@walla.com" } in the body of the POST req
  • https://project-benor-2021.herokuapp.com/vehicles/vehiclespost
    To add vehicles in the body of the POST req After login
    { "name": "car name",
    "info": "info about",
    "category": "category",
    "img_url": "img",
    "price":"price"
    }

  • https://project-benor-2021.herokuapp.com/vehicles/iddel
    To DELETE vehicles after login and get token
    for ex: /vehicles/61a89230b81cb6319c27368b

  • https://project-benor-2021.herokuapp.com/vehicles/:idEdit
    To Update vehicles after login and get token
    for ex: /vehicles/61a89230b81cb6319c27368b
